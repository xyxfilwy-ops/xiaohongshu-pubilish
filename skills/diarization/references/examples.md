# Diarization 示例

## 示例1：专家观点演变分析

```bash
/diarization
  --target "Ilya Sutskever"
  --sources "./AI专家信源/2026-04-10.md,./AI专家信源/2026-04-11.md"
  --focus 观点
  --timeline true
```

| 本方观点 | 对立观点 | 对立方 | 分析 |

## 分析师判断

### 核心洞察
Ilya的观点转变标志着AI领域的重要拐点：单纯增加算力和数据的路径遇到瓶颈，行业需要寻找新的突破方向。

### 深层分析
Ilya离开OpenAI创办SSI，其"Scaling触顶"观点可能反映了与OpenAI主流路线的战略分歧。他强调的"智力竞赛"暗示着对算法创新和架构突破的重视。

### 风险提示
- 观点可能受个人立场影响（离职创业背景）
- "触顶"判断尚需更多实证支持

## 缺口标注

### 信息缺失
- 对"智力竞赛"具体方向的具体说明
- 对AGI时间线的最新预测

### 建议补充信源
- SSI公司官方声明
- Ilya在学术会议的最新发言

## 示例3：主题追踪（AI Agent发展）

### 输入

```bash
/diarization
  --target "AI Agent发展趋势"
  --sources "./AI专家信源/2026-04-10.md,./AI专家信源/2026-04-11.md"
  --focus 预测
```

### 输出

```markdown
# AI Agent发展趋势 结构化档案

## 基本信息
- **主题**：AI Agent发展趋势
- **时间范围**：2026-04-10 至 2026-04-11
- **信源数量**：2
- **分析日期**：2026-04-13

## 核心预测矩阵

| 预测 | 提出者 | 依据 | 时间 | 可信度 |
|------|--------|------|------|--------|
| 2026年是Agent工程分水岭 | Harrison Chase | LangGraph实践 | 2026-01 | 高 |
| Agent从Talkers走向Doers | Harrison Chase/红杉 | 技术拐点分析 | 2026-01 | 高 |
| Long Horizon Agents是关键 | 多位专家 | 技术发展趋势 | 2026-04 | 高 |

## 共识与分歧

### 共识
- 2026年是Agent工程的关键年份
- 需要从对话能力转向执行能力
- 需要更好的长期规划和记忆能力

### 分歧
- 路径：框架优先 vs 模型能力优先
- 时机：已成熟 vs 还需突破

## 时间线

```
2025年 → Agent框架涌现（LangChain、LangGraph）
    ↓
2026年初 → "Doers元年"判断
    ↓ 
2026-04 → 企业级部署成为主流

趋势：从实验到生产，从Talkers到Doers
```

## 分析师判断

AI Agent领域正在经历从"能对话"到"能做事"的转变。Harrison Chase提出的"Agent Harness"概念（有主见的软件外壳）可能是关键突破点。LangChain融资成为独角兽验证了这个赛道的价值。
```
