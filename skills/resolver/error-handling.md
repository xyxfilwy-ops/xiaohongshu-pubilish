# Resolver失败处理指南

> 当技能执行失败时，自动加载本指南进行错误处理

---

## 一、失败判定标准

### 1.1 显式失败

| 错误类型 | 判定条件 | 处理方式 |
|----------|----------|----------|
| 工具调用失败 | 返回error字段 | 重试或降级 |
| 超时 | 执行时间>超时限制 | 重试或跳过 |
| 权限错误 | 401/403错误码 | 请求授权 |

### 1.2 隐式失败

| 判定条件 | 说明 |
|----------|------|
| 失败次数 > 3 | 触发自进化 |
| 用户明确否定 | "不对"、"不是这样"、"不满意" |
| 结果质量低 | 输出与预期不符超过50% |

## 二、重试策略

### 2.1 指数退避重试

```python
retry_config = {
    "max_attempts": 3,
    "base_delay": 1,      # 基础延迟(秒)
    "max_delay": 10,      # 最大延迟(秒)
    "backoff_factor": 2   # 退避因子
}

# 重试间隔: 1s → 2s → 4s
```

### 2.2 分层降级策略

```
┌─────────────────────────────────────────────────────────────┐
│                    分层降级执行流程                          │
├─────────────────────────────────────────────────────────────┤
│  Level 1: 首选方案（高精度）                                │
│     ↓ 失败                                                    │
│  Level 2: 降级方案（中等精度）                               │
│     ↓ 失败                                                    │
│  Level 3: 保底方案（基础功能）                              │
│     ↓ 失败                                                    │
│  触发自进化                                                    │
└─────────────────────────────────────────────────────────────┘
```

## 三、常见失败场景与处理

### 3.1 技能未找到

```
问题：用户输入无法匹配到任何技能
原因：触发词不在路由表中
处理：
  1. 加载 skill-assistant
  2. 自动搜索匹配技能
  3. 若找到，更新 resolver-config.md
  4. 若未找到，提示用户
```

### 3.2 参数解析失败

```
问题：用户参数不符合技能要求
原因：参数格式错误/缺少必填参数
处理：
  1. 读取技能的 SKILL.md
  2. 提取参数说明
  3. 向用户询问缺失参数
  4. 更新 skill-descriptions.md（如果需要）
```

### 3.3 工具调用失败

```
问题：bash/python/其他工具调用失败
原因：网络、环境、权限问题
处理：
  1. 记录错误日志
  2. 尝试替代方案
  3. 若多次失败，触发自进化
```

### 3.4 用户不满意

```
问题：用户明确表示结果不满意
原因：可能是风格、质量、方向问题
处理：
  1. 加载 improvement-workflow.md
  2. 分析不满意原因
  3. 迭代改进
  4. 若反复失败，触发自进化
```

## 四、自进化触发条件

### 4.1 触发阈值

| 指标 | 阈值 | 说明 |
|------|------|------|
| 连续失败次数 | > 3 | 同类型任务连续失败 |
| 失败率 | > 30% | 最近10次任务失败超过3次 |
| 用户负反馈 | 2次+ | 连续2次收到用户否定 |

### 4.2 自进化流程

```python
def trigger_self_evolution(failure_info):
    # 1. 收集失败案例
    cases = collect_failure_cases()
    
    # 2. 分析失败原因
    analysis = analyze_root_cause(cases)
    
    # 3. 生成改进建议
    suggestions = generate_improvements(analysis)
    
    # 4. 更新技能文件
    update_skill_files(suggestions)
    
    # 5. 通知主agent
    notify_main_agent("技能已自动优化")
```
