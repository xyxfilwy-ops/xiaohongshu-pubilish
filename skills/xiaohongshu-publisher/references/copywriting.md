# 爆款文案写作指南

## 核心目标

**让用户看完想收藏、想评论、想分享**

小红书文案 = 标题 + 正文 + 话题

---

## 一、爆款标题系统

### 8大标题钩子公式

#### 1. 数字冲击型（CTR最高）

**特征**：标题里有具体数字，数字越反常识越好

**公式**：
- `{数字} + {时间} + {结果}`
- `只用{数字}招 搞定{痛点}`
- `{数字}%的人都不知道的{秘密}`

**示例**：
- "3天瘦5斤 我只做了这一件事"
- "只用2招 搞定拖延症"
- "90%的人学习方法都是错的"
- "15分钟 顶别人3小时"
- "坚持30天 我的变化吓到自己"

**适用**：有具体数据/时间/步骤的内容

---

#### 2. 对比冲突型（制造认知落差）

**特征**：A vs B 或 以为X其实Y

**公式**：
- `{错误做法} vs {正确做法}`
- `你以为的{A} 其实是{B}`
- `{表象} 背后的{真相}`

**示例**：
- "假努力 vs 真进步 差在哪"
- "你以为在学习 其实在表演"
- "看起来很忙 其实在原地踏步"
- "收藏≠学会 这才是真相"

**适用**：纠正误区/颠覆认知的内容

---

#### 3. 痛点扎心型（戳中焦虑）

**特征**：直接说出读者的痛，让人觉得"说的就是我"

**公式**：
- `为什么你{痛点}`
- `{痛点} 问题出在这`
- `别再{错误行为}了`

**示例**：
- "为什么你总是学了就忘"
- "越努力越焦虑 问题出在这"
- "别再无效社交了 累的是你自己"
- "道理都懂 就是做不到？"

**适用**：解决具体问题/焦虑的内容

---

#### 4. 极端结论型（一句话炸开）

**特征**：像电影台词，短、狠、有力

**公式**：
- `{极端结论}`
- `{反常识断言}`

**示例**：
- "学霸从来不靠努力"
- "自律是个骗局"
- "你的勤奋 正在毁掉你"
- "越早知道越好的人生真相"

**适用**：观点鲜明/颠覆性结论的内容

---

#### 5. 方法清单型（承诺干货）

**特征**：明确告诉读者"我要教你几招"

**公式**：
- `{数字}个{方法}让你{结果}`
- `{场景}必备的{数字}个技巧`
- `一文讲透{主题}`

**示例**：
- "5个习惯让你越来越聪明"
- "考前必备的3个提分技巧"
- "一文讲透费曼学习法"
- "高效学习的4个底层方法"

**适用**：教程/攻略/方法论内容

---

#### 6. 悬念好奇型（不点进来就难受）

**特征**：故意不说完，留个钩子

**公式**：
- `{现象} 原因竟然是……`
- `我发现了{秘密}`
- `{意外结果} 只因为做了这件事`

**示例**：
- "学霸的秘密 原来是这个"
- "我发现了一个学习bug"
- "成绩暴涨 只因为改了一个习惯"
- "后悔没早点知道的学习方法"

**适用**：有意外发现/转折的内容

---

#### 7. 身份认同型（圈定目标人群）

**特征**：明确说"这是给谁看的"

**公式**：
- `{身份}必看`
- `给{身份}的{数字}条建议`
- `{身份}都在偷偷做的事`

**示例**：
- "大学生必看 这样学效率翻倍"
- "给职场新人的5条忠告"
- "学霸都在偷偷用的方法"
- "内向的人 请逼自己做这件事"

**适用**：针对特定人群的内容

---

#### 8. 代价警告型（不看会后悔）

**特征**：告诉你不做的代价

**公式**：
- `再不{行动} 就来不及了`
- `{错误} 正在毁掉你的{重要事物}`
- `你还在{错误行为}？醒醒吧`

**示例**：
- "再不改掉这个习惯 你会一直原地踏步"
- "这个错误正在毁掉你的学习效率"
- "你还在死记硬背？醒醒吧"
- "别等来不及了才后悔"

**适用**：纠错/警示类内容

---

### 标题写作铁律

#### 必须做到

- 口语化：像朋友发微信，不像写论文
- 15字以内：手机屏幕一行能看完
- 有情绪温度：读完心里"咯噔一下"
- 有信息增量：读完知道这篇讲什么
- 用中文标点：禁止英文标点

#### 绝对禁止

- ❌ 首先/其次/总之/综上所述
- ❌ 不是……而是……
- ❌ 想象一下……
- ❌ 在这个……的时代
- ❌ 非常重要/至关重要/不可或缺
- ❌ 破折号（全部用逗号代替）
- ❌ 空洞承诺（"改变你的人生"）
- ❌ 超过15字

#### 评分标准

| 特征 | 分数 |
|------|------|
| 有数字 | +2分（数字是CTR最强武器） |
| 有冲突 | +2分（矛盾制造好奇） |
| 有情绪 | +1分（情绪驱动点击） |
| 12字以内 | +1分（越短越狠） |
| 有AI味 | -5分（直接淘汰） |

## 二、正文文案系统

### 正文结构

```
[钩子段：50字以内]
↓
[核心价值点：3-5个要点]
↓
[行动引导：收藏/关注/评论]
```

### 钩子段写法

**目的**：前3行定生死，决定用户是否继续看

**类型**：
1. **极端台词**："三天了，你没和真人好好说过一句话。"
2. **场景代入**："凌晨两点，你又在刷手机，明天还有早会。"
3. **数据冲击**："测了一下，这个方法效果好了10倍。"
4. **痛点扎心**："说真的，这个坑我踩了整整两年。"

### 核心内容写法

**原则**：
- 每段不超过3行
- 用数字和符号分割
- 关键词加【】或「」
- 有干货有共鸣

**示例**：

```
说真的，这个方法我用了整整3个月才摸索出来。

核心就3点：

1️⃣ 【先做再学】
别等准备好了再开始，边做边学效率高10倍

2️⃣ 【输出倒逼输入】
学完就写、就讲、就用，不然白学

3️⃣ 【间隔重复】
今天学完，明天复习，一周后再复习，记得牢
```

### 行动引导

**收藏引导**：
- "码住！建议先收藏再看"
- "干货太多了 建议收藏慢慢消化"
- "这个一定要存下来 以后用得到"

**互动引导**：
- "你是哪种类型？评论区告诉我"
- "还有什么想知道的？评论区见"
- "试过这个方法吗？效果怎么样？"

**关注引导**：
- "关注我，每天分享[领域]干货"
- "持续更新中，关注不迷路"

## 三、话题标签系统

### 话题选择策略

每篇笔记选 5-8 个话题：

| 类型 | 数量 | 说明 | 示例 |
|------|------|------|------|
| 核心话题 | 1-2个 | 与内容直接相关的大话题 | #学习方法 #自律 |
| 细分话题 | 1-2个 | 更精准的垂直话题 | #费曼学习法 #间隔重复 |
| 场景话题 | 1个 | 使用场景相关 | #备考 #考研 |
| 人群话题 | 1个 | 目标人群标签 | #大学生 #职场新人 |
| 情绪话题 | 1个 | 情绪共鸣类 | #自我提升 #干货分享 |

### 话题热度查询

发布前搜索话题，优先选择：
- 搜索量大（>1000万浏览）
- 近期有热度增长
- 与内容高度相关

### SEO关键词植入

**原则**：自然植入，不刻意堆砌

**位置**：
- 标题：必须包含主关键词
- 正文第一段：包含主关键词
- 正文中间：包含2-3个长尾词
- 话题标签：包含核心关键词

## 四、去AI味完全指南

### AI味检测词库

以下词汇/句式出现 = AI味 = 淘汰：

**论文感连接词**：
- 首先、其次、再次、最后
- 总之、综上所述、总而言之
- 此外、另外、与此同时
- 值得注意的是、需要指出的是

**AI标志性句式**：
- 不是……而是……
- 想象一下……
- 你有没有想过……
- 让我们来看看……
- 在这个……的时代
- 随着……的发展

**空洞修饰词**：
- 非常重要、至关重要、不可或缺
- 毋庸置疑、众所周知、不言而喻
- 显而易见

**鸡汤套话**：
- 相信自己、你值得拥有
- 未来可期、一切都会好的
- 只要努力就一定能、坚持就是胜利

### 口语化替换词典

| AI味 | 人味 |
|------|------|
| 非常 | 超级/特别 |
| 十分 | 特别/真的 |
| 因此 | 所以/所以说 |
| 然而 | 但是/但问题是 |
| 尽管 | 虽然 |
| 此外 | 还有啊/对了 |
| 例如 | 比如/比方说 |
| 即 | 就是 |
| 获取 | 拿到/搞到 |
| 实现 | 做到 |
| 提升 | 拉高/变强 |
| 优化 | 改进/调整 |
| 显著 | 明显/很大 |
| 探索 | 试试/摸索 |
| 旅程 | 过程 |
| 赋能 | 帮到 |
| 底层逻辑 | 核心原因/根本原因 |
| 认知升级 | 想明白了 |
| 深度思考 | 好好想想 |
| 高效 | 快 |
| 低效 | 慢 |

### 写作风格要求

**像朋友发微信语音转文字**：
- 用"这玩意儿"、"说真的"、"挺有意思"
- 短句：每句不超过15字
- 用"你"称呼读者
- 反问句增加互动：是不是？对吧？

**有情绪温度**：
- 扎心感：读者感觉被"看透了"
- 代价感：不行动会失去什么
- 反转感：原来是这样
- 爽感：终于有人说明白了
- 温暖感：有人懂我

**生活化类比**：
- "全局注意力就像写作文，每写一个字都要从头复习一遍"
- "间隔重复就像浇花，每天浇一点比一次性灌一桶水活得久"
- "选择性忽略就像坐在洗脚城大厅，把噪音当背景音"

## 五、金句写作技巧

### 金句标准

- 10-20字
- 能独立截图传播
- 让人想发朋友圈
- 通过"截图测试"

### 金句公式

**对比型**：
- "忙不是价值，可复用的方法才是资产"
- "努力不是目的，进步才是"

**反转型**：
- "你以为的捷径，其实是最远的弯路"
- "最慢的方法，往往是最快的"

**隐喻型**：
- "无条件的满足，是最温柔的毒药"
- "收藏夹是知识的坟墓"

**真相型**：
- "你不是不够努力，你是方向反了"
- "道理懂100遍，不如做1遍"

## 六、文案质检清单

发布前检查：

| 检查项 | 标准 |
|--------|------|
| 标题字数 | ≤15字 |
| 标题钩子 | 命中8大公式之一 |
| AI味检测 | 无禁用词/句式 |
| 口语化 | 有"你"、有反问、有口语词 |
| 段落长度 | 每段≤3行 |
| 金句数量 | 至少1句可截图的 |
| 话题数量 | 5-8个 |
| 行动引导 | 有收藏/评论/关注引导 |
| SEO关键词 | 标题+正文包含 |
