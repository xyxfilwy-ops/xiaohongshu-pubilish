## 改进建议

### 1. 超时 (优先级: P1)

**问题**: 连续2次出现超时相关问题
**建议方案**: 添加超时重试机制，失败后自动重试1-2次
**规则类型**: 错误处理

**示例案例**:
- 图片生成超时，提示request timeout 1204
- 图片超时了，API返回1204错误


```markdown
### 超时处理规则
- **触发条件**: API响应超时
- **处理方式**: 
  1. 首次失败等待2秒重试
  2. 第二次失败等待5秒重试
  3. 第三次失败返回明确错误信息
- **示例**:
  ```
  if "timeout" in str(error):
      retry_with_backoff(max_retries=2)
  ```
```
---
### 2. API错误 (优先级: P1)

**问题**: 连续1次出现API错误相关问题

**规则类型**: 错误处理

**示例案例**:
- 图片超时了，API返回1204错误


```markdown
### API限流处理
- **触发条件**: 429错误或rate limit相关
- **处理方式**:
  1. 解析Retry-After头
  2. 等待指定时间后重试
  3. 记录连续触发次数
```
