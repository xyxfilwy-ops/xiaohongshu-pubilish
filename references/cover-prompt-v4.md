# 封面图生成指南 v4.0（RedInk风格简化版）

## 核心理念

**简单就是好**：不使用复杂的英文分层结构，用简单的中文描述让AI理解。

## 封面图生成模板

```
请生成一张小红书风格的封面图，尺寸1080x1440像素。

【标题内容】
{标题文字}

【设计要求】
1. 整体风格：小红书爆款封面，清新精致有设计感
2. 背景色：{用中文描述颜色，如"深灰蓝渐变"、"暖米色"、"浅粉渐变"}，不要出现颜色代码
3. 标题排版：
   - 标题要大、要醒目，占画面50%以上
   - 字体清晰，一眼能看清
   - 可以把关键词放大突出
4. 装饰元素：简单点缀，不要复杂
5. 可选：在角落放一个小柴犬卡通形象，橘黄色毛发，白色脸，戴着眼镜，表情好奇

【注意事项】
- 标题必须清晰可读，不能模糊
- 不要在图片上显示颜色代码或技术参数
- 装饰要克制，标题是主角
```

## Doggie IP 简化描述

**为什么用Doggie**：品牌识别 + 情感连接 + 差异化

**简化描述**：
```
一只可爱的Q版柴犬卡通形象：
- 橘黄色毛发，白色脸蛋花纹
- 圆圆的脸，三角形的立耳朵
- 黑豆眼，小黑鼻，粉色腮红
- 穿着{服装}，表情{表情}
- 大小：占图片高度20%左右
- 位置：右下角或左下角
```

**服装对应**：
- AI/科技类 → 白衬衫+针织背心+圆眼镜（认真可爱）
- 生活/日常类 → 奶白色卫衣（温暖治愈）
- 情感/心理类 → 粉色毛衣（温柔）
- 职场/事业类 → 小西装+领结（专业）

## 品类背景色描述（中文）

| 品类 | 背景色描述 |
|------|-----------|
| AI/科技 | 深灰蓝渐变，科技感，带一点霓虹光晕 |
| 生活/日常 | 暖米色，柔和温馨 |
| 情感/心理 | 浅粉渐变，温柔治愈 |
| 职场/事业 | 深蓝渐变，专业沉稳 |
| 健身/运动 | 活力橙渐变，能量感 |
| 美妆/时尚 | 莫兰迪粉色，精致高级 |

## 避坑指南

❌ **错误写法**：
```
Base color: #1A1A2E → #16213E
```
这会被AI当作文字渲染在图片上！

✅ **正确写法**：
```
背景色：深灰蓝渐变，从深色到稍浅的蓝色，科技感氛围
```

## 实际示例

**标题**：SBTI刷屏了，我连夜做了个情绪版

**Prompt**：
```
请生成一张小红书风格的封面图，尺寸1080x1440像素。

【标题内容】
SBTI刷屏了
我连夜做了个情绪版

【设计要求】
1. 整体风格：小红书爆款封面，清新精致有设计感
2. 背景色：深灰蓝渐变，科技感，带一点霓虹光晕
3. 标题排版：
   - "SBTI刷屏了"放上面，字号大
   - "我连夜做了个情绪版"放下面，字号稍小
   - "情绪版"可以用亮色突出
4. 装饰元素：简单的发光点、线条
5. 右下角放一只Q版柴犬：橘黄毛色，白脸，戴圆眼镜穿背心，表情认真可爱，占画面20%高度

【注意事项】
- 标题必须清晰可读
- 不要显示颜色代码或技术参数
- 柴犬是配角，不要太大
```
