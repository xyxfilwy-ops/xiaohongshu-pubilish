# 内容图生成指南 v4.0（RedInk风格简化版）

## 核心理念

**简单就是好 + 参考图机制**：封面生成后，作为参考图传给后续内容页，保持风格一致。

## 内容图生成模板

```
请生成一张小红书风格的内容页图片，尺寸1080x1440像素。

【页面内容】
{正文内容}

【页面信息】
- 页面类型：内容页（第{X}页，共{Y}页）
- 参考风格：请参考封面图的风格，保持统一

【设计要求】
1. 整体风格：与封面保持一致，小红书爆款图文风格
2. 背景色：{与封面相同的背景色描述}
3. 文字排版：
   - 正文清晰可读，字号适中
   - 每段之间有呼吸感
   - 关键词可以加粗或用亮色
4. 页码：右下角显示"{X}/{Y}"，小字，透明度50%
5. 可选：在角落放一个小柴犬卡通形象，与封面风格一致

【注意事项】
- 文字必须清晰可读
- 不要显示颜色代码
- 保持与封面风格统一
```

## 参考图传递机制（关键！）

```
Step 1: 生成封面图
→ 保存封面图路径

Step 2: 生成内容页时
→ 在image_generate工具中传入 ref_images=[封面图路径]
→ 在prompt中强调"请参考封面图的风格"
```

## 内容页类型与Doggie动作

| 页面类型 | 内容特点 | Doggie动作 | 位置 |
|----------|----------|------------|------|
| 开头/钩子 | 抓注意力 | 好奇张望 | 左下角 |
| 问题/痛点 | 揭示问题 | 捂嘴惊讶 | 右下角 |
| 分析/解释 | 详细说明 | 挠头思考 | 右下角 |
| 方法/步骤 | 给出方案 | 竖指认真 | 右下角 |
| 案例/对比 | 实例展示 | 左右比划 | 中下 |
| 总结/收尾 | 归纳要点 | 挥手告别 | 底部中央 |

## 实际示例

**第1页 - 钩子页**

**Prompt**：
```
请生成一张小红书风格的内容页图片，尺寸1080x1440像素。

【页面内容】
SBTI是什么？
就是一个超火的人格测试
朋友圈刷屏了

【页面信息】
- 页面类型：内容页（第1页，共5页）
- 参考风格：请参考封面图的风格，保持统一

【设计要求】
1. 整体风格：与封面保持一致，深灰蓝渐变背景
2. 背景色：深灰蓝渐变，科技感
3. 文字排版：
   - "SBTI是什么？"是标题，字号大
   - 下面两行是正文，字号适中
   - 行距要舒服
4. 页码：右下角显示"1/5"，小字
5. 左下角放一只Q版柴犬：与封面风格一致，好奇张望的表情

【注意事项】
- 文字必须清晰可读
- 不要显示颜色代码
- 保持与封面风格统一
```

## 批量生成流程

```
1. 先生成封面图
   - 保存路径：./xiaohongshu-output/封面.png

2. 内容页1
   - ref_images: ["./xiaohongshu-output/封面.png"]
   - 强调参考封面风格

3. 内容页2
   - ref_images: ["./xiaohongshu-output/封面.png"]
   - 继续参考封面风格

4. ...以此类推
```

## 避坑指南

❌ **错误写法**：
```
每页独立生成，不传参考图
结果：每页风格不一样，视觉混乱
```

✅ **正确写法**：
```
封面生成后 → 作为参考图传给所有内容页
结果：风格统一，视觉专业
```
